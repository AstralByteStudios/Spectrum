# Proyecto: Galería de Fondos de Pantalla

Este proyecto es una galería de fondos de pantalla interactiva que permite a los usuarios ver, filtrar y descargar fondos de pantalla desde Firebase Storage. Los fondos están organizados por categorías y se muestran con un diseño atractivo.

## Características

- Visualización de imágenes de fondos de pantalla desde Firebase Storage.
- Categorías de fondos de pantalla para una mejor organización.
- Funcionalidad de descarga para cada fondo de pantalla.
- Filtros interactivos para seleccionar categorías.
- Modal para ver imágenes a tamaño completo.

## Tecnologías Utilizadas

- **HTML**: Estructura del contenido.
- **CSS**: Estilos de la página web.
- **JavaScript**: Funcionalidades dinámicas, como cargar imágenes, mostrar modales y aplicar filtros.
- **Firebase**: Almacenamiento de los fondos de pantalla (Firebase Storage).
- **Vite**: Herramienta de construcción para proyectos frontend (para un desarrollo más rápido y eficiente).
